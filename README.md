## 📖 מה זה שיחבוט?

שיחבוט מאפשר לך לצפות בשיחה חיה בין **מספר מודלי AI שונים** על נושא שתבחר — או שהמודלים יציעו לך נושא בעצמם. כל מודל מגיב על דברי המודלים האחרים, יוצר דיון אמיתי ומעניין. כל התוכן מוצג **בעברית בלבד**.

הפרויקט זמין גם כ**אפליקציית EXE לווינדוס** וגם כ**אתר אינטרנט** דרך Vercel.

## ✨ פיצ'רים מרכזיים

### 🗣️ שיחות בין מודלים
- בחר **2 מודלים ומעלה** לשיחה מתוך מעל 40 מודלים זמינים
- כל מודל מגיב על דברי המודלים האחרים בצורה טבעית
- **עברית בלבד** — הוראות מחמירות מבטיחות שאף מודל לא יחרוג לשפה אחרת
- תמיכה בנושאי שיחה חופשיים — כל מה שתכתוב

### 🤖 מאגר מודלים עשיר
מעל 40 מודלים מחולקים לקטגוריות:

| קטגוריה | דוגמאות |
|---|---|
| **Llama** | Llama 3.3 70B, Llama 3.1 405B 🔥 |
| **Google** | Gemini 3.1 Flash Lite 🚀, Gemini 2.5 Flash 🔥 |
| **Qwen** | Qwen3 235B MoE 🔥, QwQ 32B Reasoning 🧠 |
| **DeepSeek** | DeepSeek R1 0528 🔥, DeepSeek V3 |
| **Microsoft** | Phi-4 Reasoning+, Phi-4 Mini |
| **ועוד** | Gemma 2, Command A, Kimi K2 |

> מודלי HuggingFace פועלים דרך HF Inference API — מודלי Gemini דרך Google Generative AI API

### 🎭 אישיויות מותאמות
45 אישיויות מוכנות לבחירה לכל מודל:
- **מקצועיות**: פילוסוף, מדען, עורך דין, רופא, בלש, היסטוריון...
- **ישראלי**: חייל, סבתא מרוקאית, הייטקיסט תל אביבי, שייח בדואי...
- **יצירתי**: מוזיקאי, סופר, שף גאוותן, קוסם...
- **אישיות מותאמת אישית** — כתוב בעצמך

### 💻 מצב מפתח
מצב ייעודי לפיתוח תוכנה שיתופי בין המודלים:
- תאר מה ברצונך לפתח בשדה חופשי
- **עלה קוד קיים** לעריכה ושיפור
- כל מודל מקבל תפקיד אוטומטי:
  - 👨‍💻 מפתח ראשי — כותב את הקוד
  - 🔍 מבקר קוד — מציע שיפורים
  - 🧪 בודק QA — מאתר באגים
  - 🏗️ ארכיטקט — מבנה ועיצוב
  - ⚡ מייעל ביצועים
  - 🛡️ מומחה אבטחה
  - 📝 כותב תיעוד

### 📎 העלאת קבצים

- **כל עוד אתה מקליד — המודלים מחכים!**

## 🚀 פריסה ל-Vercel

### מבנה הקבצים
```
repository/
├── api/
│   └── index.py        ← Flask backend
├── index.html
├── style.css
├── index.js
├── vercel.json
└── requirements.txt
```

### שלבי הפריסה
1. **העלה לGitHub** — צור repo חדש והעלה את כל הקבצים
2. **Vercel** — היכנס ל-[vercel.com](https://vercel.com), "Add New Project", בחר את ה-repo
3. לחץ **Deploy** — Vercel מזהה הכל אוטומטית!

### משתני סביבה (אופציונלי)
| משתנה | תיאור |
|---|---|
| `HF_API_KEY` | מפתח HuggingFace קבוע (אופציונלי) |

## 🔑 מפתחות API

### HuggingFace (חובה לרוב המודלים)
1. היכנס ל-[huggingface.co/settings/tokens](https://huggingface.co/settings/tokens)
2. צור token חדש עם הרשאת **Inference**
3. הכנס בחלון ה-API בתוכנה

### Google Gemini (אופציונלי — למודלי Gemini ולפודקאסט)
1. היכנס ל-[aistudio.google.com](https://aistudio.google.com)
2. צור API key
3. הכנס בחלון ה-API בתוכנה
